## 请求处理流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        单次工具调用流程                                       │
└─────────────────────────────────────────────────────────────────────────────┘


用户请求
    │
    │  "导航到 example.com"
    ▼
┌─────────────────┐
│  Client.parse   │
│  解析用户输入    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Tool Selection │
│  选择工具:      │
│  browser_navigate
└────────┬────────┘
         │
         │ 构造 JSON-RPC 请求
         │ {
         │   "jsonrpc": "2.0",
         │   "id": 1,
         │   "method": "tools/call",
         │   "params": {
         │     "name": "browser_navigate",
         │     "arguments": {
         │       "url": "https://example.com"
         │     }
         │   }
         │ }
         ▼
┌─────────────────┐
│  stdio Write    │
│  写入标准输出    │ ─────► Server stdin
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Server.parse   │
│  解析 JSON-RPC  │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Tool Registry  │
│  查找工具处理器  │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Handler.call   │
│  调用处理函数    │
└────────┬────────┘
         │
         │ ┌─────────────────────────────────────┐
         │ │  browser_navigate handler           │
         │ │  1. 检查浏览器状态                   │
         │ │  2. 调用 page.goto(url)             │
         │ │  3. 等待页面加载                     │
         │ │  4. 获取页面标题和 URL               │
         │ └─────────────────────────────────────┘
         │
         ▼
┌─────────────────┐
│  Format Result  │
│  格式化结果      │
│  [TextContent]  │
└────────┬────────┘
         │
         │ 构造 JSON-RPC 响应
         │ {
         │   "jsonrpc": "2.0",
         │   "id": 1,
         │   "result": {
         │     "content": [...],
         │     "isError": false
         │   }
         │ }
         ▼
┌─────────────────┐
│  stdio Write    │
│  写入标准输出    │ ─────► Client stdin
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Client.display │
│  显示结果给用户  │
│  "✅ 已导航到..." │
└─────────────────┘
```

## 数据传输格式

### 请求格式

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "tools/call",
  "params": {
    "name": "browser_navigate",
    "arguments": {
      "url": "https://example.com",
      "wait_until": "networkidle"
    }
  }
}
```

### 响应格式 (成功)

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "✅ 已导航到: https://example.com\n标题: Example Domain"
      }
    ],
    "isError": false
  }
}
```

### 响应格式 (错误)

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "❌ 错误: 浏览器未启动"
      }
    ],
    "isError": true
  }
}
```

### 图片响应

```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "✅ 截图成功"
      },
      {
        "type": "image",
        "data": "iVBORw0KGgoAAAANSUhEUgAA...",  // base64 encoded PNG
        "mimeType": "image/png"
      }
    ],
    "isError": false
  }
}
```


## 安全考虑

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           安全机制                                            │
└─────────────────────────────────────────────────────────────────────────────┘


1. 进程隔离
   ┌──────────┐     stdio     ┌──────────┐
   │  Client  │◄─────────────►│  Server  │
   │  (Python)│    (受限)     │ (Python) │
   └──────────┘               └──────────┘
   
   • 通过 stdio 通信，无网络暴露
   • Server 崩溃不影响 Client
   • 可通过操作系统权限控制

2. 输入验证
   • 所有工具参数都有 JSON Schema 验证
   • URL 格式检查
   • JavaScript 代码沙箱 (通过 Playwright 隔离)

3. 资源限制
   • 浏览器启动参数可控
   • 可限制无头模式/有界面模式
   • 窗口大小限制

4. 状态隔离
   • 每个 Context 独立 Cookie/Storage
   • 无跨会话数据泄露
   • 关闭后资源自动清理
```
